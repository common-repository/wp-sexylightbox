Plugin that integrates sexylightbox2 into wordpress.

==  Description ==

This plugin will add SexyLightBox to all the images in your blog, as long as they're linking to the image file, it's not necessary to add "rel=lightbox" anywhere, the plugin does it for you.

== Installation ==

1. Extract the files 
2. Upload the folder /wp-sexylightbox to /wp-content/plugins
3. Activate the plugin
4. That's it! Now you can change colors from SexyLightBox menu in the options tab
